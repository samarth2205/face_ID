


# face_ID

Aim is to create face identification system using Tensorflow framework


### Applications
* Attendace System at school/office
* Crowd Dynamics
* Surveillance System


### dependencies

* Python => 3.6.*
* Tensorflow (latest)
* Numpy

pipeline:-

Image -> Face detection -> Cropped face -> Face Recognition -> Descriptor(128D) -> Matching -> Name.
